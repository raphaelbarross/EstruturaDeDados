# EstruturaDeDados

Repositório destinado ao estudo e prática de **Estruturas de Dados**, com implementações simples e didáticas em **C**.

## Estrutura do projeto

- **Arvore**  
  Implementações relacionadas a árvores.

- **Fila**  
  Estrutura de dados do tipo fila (FIFO).

- **Pilha**  
  Estrutura de dados do tipo pilha (LIFO).

- **Lista**  
  Implementações de listas (sequenciais e/ou encadeadas).

- **EstruturasGerais**  
  Conceitos e estruturas básicas de apoio.

##  Objetivo

- Praticar lógica de programação  
- Entender o funcionamento interno das estruturas de dados  
- Servir como material de consulta e evolução acadêmica

##  Linguagem utilizada

- C

##  Observações

Este repositório tem **finalidade educacional**.  
Os códigos priorizam clareza e aprendizado, não otimização avançada.
